## 1. Architectural Frame

- **Tier 1 (Source):** `.cdx` documents are the only normative authority.
- **Tier 2 (Projection):** Markdown files generated by the projector from validated Tier 1 sources. Targets are enumerated by `TargetFileKind` (see §2).
- **Direction of Flow:** Tier 1 → Tier 2 only. Output never feeds back into source.
- **Zero Invention:** The schema is closed. No implicit defaults, no foundry discretion.

### Imports

- `paperhat-text` (namespace `text`): provides `text:Paragraph`.
- `paperhat-grammar` (namespace `grammar`): provides `grammar:Grammar` and `grammar:ProductionRule`.

### Constraints

- `path-normalization`: leading slash forbidden; `..` segments forbidden; repeated slashes forbidden.
- `match-mode-root`: content must be exactly `.`.
- `match-mode-literal`: must not end with `/`; wildcards forbidden.
- `match-mode-prefix`: must end with `/`; wildcards forbidden.
- `match-mode-glob`: tokens limited to literals, `/`, `*`, `?`, `**`.

## 3. Evaluation Rules

1. Highest precedence wins.
2. `$OutOfScope` shadows `$InScope` on precedence tie.
3. No match defaults to `$OutOfScope`.
4. `$Root`, `$Literal`, and `$Prefix` are anchored to the directory containing the defining `.agents` file.
5. `$Glob` is the only mode allowed to express anywhere or recursive matches (`**`).
