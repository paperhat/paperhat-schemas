# Energy Measure Schema

- **Status**: NORMATIVE
- **Lock State**: LOCKED
- **Version**: 1.0.0
- **Editor**: Charles F. Munat

## Purpose

A dimension-typed Measure whose unit trait is constrained to tokens from the units-energy vocabulary. Use this instead of the generic Measure when only energy, work, heat, and power units are semantically valid.

## Allowed Units

BritishThermalUnit, BritishThermalUnitsPerHour, Calorie, Electronvolt, Erg, FootPound, FootPoundsPerSecond, Gigajoule, Gigawatt, Horsepower, Joule, Kilocalorie, Kilojoule, Kilowatt, KilowattHour, Megajoule, Megawatt, MegawattHour, MetricHorsepower, Therm, Watt, WattHour.
