# Mass Measure Schema

- **Status**: NORMATIVE
- **Lock State**: LOCKED
- **Version**: 1.0.0
- **Editor**: Charles F. Munat

## Purpose

A dimension-typed Measure whose unit trait is constrained to tokens from the units-mass vocabulary. Use this instead of the generic Measure when only mass units are semantically valid.

## Allowed Units

AtomicMassUnit, Carat, Grain, Gram, Kilogram, LongTon, MetricTon, Microgram, Milligram, Ounce, Pound, ShortTon, Stone, TroyOunce.
