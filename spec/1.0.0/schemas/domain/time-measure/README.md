# Time Measure Schema

- **Status**: NORMATIVE
- **Lock State**: LOCKED
- **Version**: 1.0.0
- **Editor**: Charles F. Munat

## Purpose

A dimension-typed Measure whose unit trait is constrained to tokens from the units-time vocabulary. Use this instead of the generic Measure when only time and duration units are semantically valid.

## Allowed Units

Day, Hour, Minute, Month, Second, Week, Year.
