# Volume Measure Schema

- **Status**: NORMATIVE
- **Lock State**: LOCKED
- **Version**: 1.0.0
- **Editor**: Charles F. Munat

## Purpose

A dimension-typed Measure whose unit trait is constrained to tokens from the units-volume vocabulary. Use this instead of the generic Measure when only volume and capacity units are semantically valid.

## Allowed Units

Barrel, Bushel, Centiliter, CubicCentimeter, CubicFoot, CubicInch, CubicMeter, CubicYard, Cup, FluidOunce, Gallon, ImperialFluidOunce, ImperialGallon, ImperialPint, ImperialQuart, Liter, Milliliter, Pint, Quart, Tablespoon, Teaspoon.
