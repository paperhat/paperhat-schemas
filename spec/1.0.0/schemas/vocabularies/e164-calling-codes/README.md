# E.164 Country Calling Codes

A closed vocabulary of international telephone country calling codes from the ITU-T E.164 recommendation.

## Purpose

Constrains `$EnumeratedToken` traits that represent telephone country codes to the set of assigned calling codes.

## Token format

Tokens use the numeric calling code (e.g., `$1`, `$44`, `$81`). Labels provide the country or region name. Some codes cover multiple territories (e.g., `$1` covers the North American Numbering Plan).

## Scope

This vocabulary includes country-level calling codes. It excludes sub-national area codes, mobile-specific prefixes, and non-geographic codes (satellite, universal international numbers).
