# ISO 3166-1 Alpha-2 Country Codes

A closed vocabulary of country and territory codes from the ISO 3166-1 alpha-2 standard.

## Purpose

Constrains `$EnumeratedToken` traits that represent countries or territories to the set of internationally recognized two-letter codes.

## Token format

Tokens use the two-letter alphabetic code (e.g., `$US`, `$GB`, `$JP`). Labels provide the short country name as used by the ISO 3166 Maintenance Agency.

## Scope

This vocabulary includes all currently assigned ISO 3166-1 alpha-2 codes, covering sovereign states, dependent territories, and special areas of geographical interest. It excludes user-assigned codes, exceptionally reserved codes, and withdrawn codes.
