# Preparations Vocabulary

- **Status**: NORMATIVE
- **Lock State**: LOCKED
- **Version**: 1.0.0
- **Editor**: Charles F. Munat

## Purpose

Defines enumerated tokens for culinary preparation techniques applied to ingredients before or during cooking.

## Token Format

Each `TokenDefinition` uses a fully spelled-out `$PascalCase` token name and a `label` carrying the lowercase past-participle form used in recipe text.

## Scope

Covers common physical preparation techniques (cutting, peeling, grating) and pre-treatment methods (blanching, brining, marinating, soaking). Does not include cooking methods (baking, frying, steaming) — those belong to a separate cooking-methods vocabulary if needed.
