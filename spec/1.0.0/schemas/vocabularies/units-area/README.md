# Units of Area

A closed vocabulary of units of measurement for area and surface extent.

## Purpose

Constrains `$EnumeratedToken` traits that represent area units. Used by the Measure schema's `unit` trait when a consuming schema restricts allowed values to area units.

## Token format

Tokens use fully spelled out PascalCase names (e.g., `$SquareMeter`, `$Hectare`, `$Acre`). Labels carry the standard symbol or abbreviation (e.g., `m2`, `ha`, `ac`).

## Scope

This vocabulary includes metric units (square millimeter through square kilometer, are, hectare) and imperial and customary units (square inch, square foot, square yard, square mile, acre). It excludes archaic or region-specific units not in common modern use.
