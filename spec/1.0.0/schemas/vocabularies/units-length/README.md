# Units of Length

A closed vocabulary of units of measurement for length, distance, and displacement.

## Purpose

Constrains `$EnumeratedToken` traits that represent length units. Used by the Measure schema's `unit` trait when a consuming schema restricts allowed values to length units.

## Token format

Tokens use fully spelled out PascalCase names (e.g., `$Kilometer`, `$NauticalMile`, `$Micrometer`). Labels carry the standard symbol or abbreviation (e.g., `km`, `nmi`, `um`).

## Scope

This vocabulary includes metric units from millimeter through kilometer, imperial and customary units (inch, foot, yard, mile), nautical and surveying units (nautical mile, fathom, chain, furlong), sub-micrometer units (nanometer, picometer, angstrom), and astronomical units (astronomical unit, light-year, parsec). It excludes archaic or region-specific units not in common modern use.
