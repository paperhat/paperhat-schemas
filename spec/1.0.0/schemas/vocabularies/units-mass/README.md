# Units of Mass

A closed vocabulary of units of measurement for mass and weight.

## Purpose

Constrains `$EnumeratedToken` traits that represent mass units. Used by the Measure schema's `unit` trait when a consuming schema restricts allowed values to mass units.

## Token format

Tokens use fully spelled out PascalCase names (e.g., `$Kilogram`, `$TroyOunce`, `$MetricTon`). Labels carry the standard symbol or abbreviation (e.g., `kg`, `ozt`, `t`).

## Scope

This vocabulary includes metric units from microgram through metric ton, imperial and customary units (ounce, pound, stone, short ton, long ton), specialized units (carat, grain, troy ounce), and the atomic mass unit. It excludes archaic or region-specific units not in common modern use.
