# Units of Time Vocabulary

- **Status**: NORMATIVE
- **Lock State**: LOCKED
- **Version**: 1.0.0
- **Editor**: Charles F. Munat

## Purpose

Defines enumerated tokens for units of time and duration.

## Token Format

Each `TokenDefinition` uses a fully spelled-out `$PascalCase` token name and a `label` carrying the standard abbreviation.

## Scope

Covers common time units from seconds through years. Does not include sub-second units (milliseconds, microseconds, nanoseconds) or calendar-specific units (quarter, decade, century).
