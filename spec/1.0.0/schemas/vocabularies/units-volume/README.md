# Units of Volume

A closed vocabulary of units of measurement for volume and capacity, including cooking measures.

## Purpose

Constrains `$EnumeratedToken` traits that represent volume units. Used by the Measure schema's `unit` trait when a consuming schema restricts allowed values to volume units.

## Token format

Tokens use fully spelled out PascalCase names (e.g., `$Milliliter`, `$CubicMeter`, `$ImperialGallon`). Labels carry the standard symbol or abbreviation (e.g., `mL`, `m3`, `imp gal`).

## Scope

This vocabulary includes metric units (milliliter, centiliter, liter, cubic centimeter, cubic meter), US customary liquid measures (fluid ounce, cup, pint, quart, gallon), imperial liquid measures (imperial fluid ounce, imperial pint, imperial quart, imperial gallon), cooking measures (tablespoon, teaspoon), cubic imperial units (cubic inch, cubic foot, cubic yard), and commodity units (barrel, bushel). It excludes archaic or region-specific units not in common modern use.
